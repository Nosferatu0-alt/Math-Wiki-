##  Contribuindo

Contribuições são bem-vindas! Se quiser adicionar um conceito:

1. Fork o repositório
2. Crie uma branch: `git checkout -b novo-conceito`
3. Adicione seu arquivo `.md` na pasta correta
4. Abra um Pull Request
