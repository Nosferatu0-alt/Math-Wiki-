## 🤝 Contribuindo

Contribuições são **muito bem-vindas**! Quer adicionar um novo conceito?

```bash
# 1. Faça um fork do repositório
# 2. Crie uma nova branch
git checkout -b feat/novo-conceito

# 3. Adicione seu arquivo .md na pasta correta
# 4. Commit e abra um Pull Request 🎉
git commit -m "✨ adiciona conceito X"
git push origin feat/novo-conceito
```
